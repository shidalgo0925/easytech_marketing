## 4. Soluciones aplicadas

| Problema | Solución |
|----------|----------|
| Cron 404 | `accio_tick.sh` → `/accio/{tenant}/tick` (default `easytech`) |
| Cron 403 | Bearer `ACCIO_API_KEY` bypass RBAC para automatización (cron, Accio Work) |
| Compatibilidad docs | Alias `POST /accio/tick` → delega a tenant default |
| Inventario obsoleto | Reescritura completa con servicios, URLs, endpoints |
| n8n import | Añadir `id` en workflow y nodos; import exitoso (inactivos) |
| Operación sin adivinar | `doctor.sh` + formato de diagnóstico obligatorio |
